# 🚀 Tech Conference Landing Page

A modern, responsive, and visually appealing landing page for **Tech Conference** built using **HTML5**, **CSS3**, and **Bootstrap 5**. The website provides conference information, event agenda, featured speakers, and an online registration form in a clean and user-friendly interface.

---

## 📖 Project Description

The **Tech Conference 2024 Landing Page** is designed to promote a technology conference by showcasing event details in an engaging and responsive layout.

The application includes:

* A responsive navigation bar
* A full-screen hero section with a call-to-action
* Conference overview
* Three-day event agenda
* Featured speakers section
* Registration form
* Responsive design compatible with desktop, tablet, and mobile devices

This project demonstrates responsive web design principles using Bootstrap components and custom CSS styling.
